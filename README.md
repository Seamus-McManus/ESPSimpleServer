# ESPSimpleServer Webserver
An easy-to-use webserver framework that runs on an ESP32! Just drop your website files and assets, upload to your board, and the rest will be handled for you!

## What it Supports
- HTTP GET, HEAD, POST, and DELETE requests
- Automatic redirects to index.html
- File uploading and deletion
- Automatic asset retrieving
- Different filesystems/partitions

  ## Credits
  This is based on the Arduino ESP32 Webserver example, but its functionality has been heavily modified in order to support automatic asset handling.
